Layout

Layout is a whiteboard for planning plays in Ultimate.

It also refers to the acts of planning, or diving for the disc.

Layout lets you view and create animations, here and embed them into your site.

Features

Layout lets you design, save and share plays and drills. You can:

  • Create plays by moving players and capturing key states
  • Animate plays -- step-by-step or continuously
  • Save/download and load plays
  • Embed plays created using layout, in any web-page!
  • Share Layouts
    1. Download your layout file
    2. Upload it somewhere on the web
    3. Enter URL to raw file

Layout is currently in "alpha" and some of the requested/planned/hope-to-add features are listed here.

Layout is best supported in the latest versions of Chromium or Google Chrome. This site is best viewed at resolutions 1024x768 and above.
